.. _installation:

Installation guide
==================


The ``gsMap`` package can be installed via pip:

.. code-block:: bash

    pip install gsMap


or via the source code:

.. code-block:: bash

    git clone https://github.com/LeonSong1995/gsMap.git
    cd gsMap
    pip install -e .

Verify the installation by running the following command:

.. code-block:: bash

    gsMap -v
